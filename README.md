## 🧠 Научный фундамент — не «подгонка по таблице»

В основе алгоритмов лежат **фундаментальные законы гидродинамики и электрических машин**, а не эмпирические корреляции.

### 1. Законы подобия центробежных насосов
При изменении частоты характеристики пересчитываются строго по классическим Affinity Laws:

<p align="center">
  <img src="https://latex.codecogs.com/svg.image?Q_2/Q_1&space;=&space;f_2/f_1" alt="Q2/Q1 = f2/f1" />
  <img src="https://latex.codecogs.com/svg.image?H_2/H_1&space;=&space;(f_2/f_1)^2" alt="H2/H1 = (f2/f1)^2" />
  <img src="https://latex.codecogs.com/svg.image?N_2/N_1&space;=&space;(f_2/f_1)^3" alt="N2/N1 = (f2/f1)^3" />
</p>

Моделируем работу на любой частоте от 35 до 200 Гц без искажений.

### 2. Поправка на вязкость через число Рейнольдса
При вязкости нефти >1,1 сП включается коррекция:
- Расчёт числа Рейнольдса потока в каналах насоса  
- По величине Re — понижающие коэффициенты подачи, напора и КПД  
- **Главное правило:** упал КПД → мощность пересчитана заново, а не взята из паспорта

### 3. Настоящая потребляемая мощность
Мощность на валу вычисляется по классической формуле, где КПД уже изменён вязкостью:

<p align="center">
  <img src="https://latex.codecogs.com/svg.image?N&space;=&space;\frac{\rho&space;\cdot&space;g&space;\cdot&space;Q&space;\cdot&space;H}{3.6\times10^6&space;\cdot&space;\eta}" alt="N = (ρ·g·Q·H) / (3.6×10⁶·η)" />
</p>

Двигатель подбирается с запасом **≥15%** от реальной, а не табличной мощности.

### 4. Деградация напора от свободного газа
Учитывается газовый фактор и давление насыщения. Напор ступени умножается на коэффициент от 1,0 (газа ≤5%) до 0,1 (газа ≥35%). Газосепаратор снижает долю газа в 4–10 раз.

### 5. Конструктивные и температурные ограничения
- Диаметр ПЭД и гидрозащиты ≤ внутренний диаметр колонны  
- Максимальная мощность насоса (прямой/плавный пуск) не превышена  
- Число секций ≤ 20  
- При температуре >120°C — только высокотемпературные исполнения

## 🚀 Основные возможности (v8.6.1)

### 🎯 Интеллектуальный рейтинг «Лучший выбор»
Каждому варианту присваивается балл (score 0–100) на основе попадания дебита в рабочую зону и близости к номинальной точке. Лучший насос помечается звёздой — вы сразу видите оптимальное решение.

### 🖱️ Интерактивные графики
- 4 режима: напор, мощность, КПД, рабочая область  
- Перетаскивайте маркер цели — дебит в исходных данных меняется автоматически, идёт перерасчёт  
- Экспорт графика в PNG одним кликом

### ⚖️ Сравнение вариантов
Отметьте несколько насосов чекбоксами — получите таблицу с параллельным сравнением подач, напоров, секций, мощности и подобранного оборудования.

### 🧮 Автоматический расчёт напора
Заполните глубину спуска, динамический уровень, буферное/затрубное давления — калькулятор сам вычислит требуемый напор с запасом. Любое изменение → мгновенный пересчёт.

### 🛡️ Валидация и подсказки
При некорректном вводе (отрицательный дебит, уровень выше глубины спуска) — мгновенные предупреждения. Цветовые бейджи на карточках: 🟢 норма, 🟠 на границе, 🔴 риск.

### 💾 Сохранение конфигураций
Все параметры можно сохранить в браузере и загрузить позже — идеально для подбора по нескольким скважинам.

### 🖨️ Печать отчёта
Чистая версия для принтера, без боковых панелей и лишних элементов.

### 🌓 Тёмная / светлая тема
Комфортная работа при любом освещении.

### 📚 Встроенный каталог оборудования
Более 80 реальных позиций: насосы 2А–7А (нирезист/ПКМ), асинхронные и вентильные ПЭД, гидрозащиты. Все характеристики из паспортных данных.

## 🧭 Типичный сценарий работы за 30 секунд

1. Введите дебит, плотность, вязкость (если нефть).  
2. Выберите диаметр обсадной колонны.  
3. Включите авторасчёт напора → укажите глубину спуска, динамический уровень, давления.  
4. При наличии газа задайте газовый фактор и тип газосепаратора.  
5. Установите частоту (например, 55 Гц) и исполнение насоса.  
6. Нажмите **«Подобрать оборудование»**.  
7. Оцените лучший вариант, при необходимости сравните несколько, перетащите точку на графике для уточнения дебита.  
8. Сохраните или распечатайте отчёт.
